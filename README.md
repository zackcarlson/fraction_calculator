# Fraction Calculator
A CLI calculator app that computes for fractions (and whole numbers, too)!

## Getting Started

Download or clone this repo.

### Prerequisites

Install the required packages: 
```npm install```
Symlink your binary file to the system path:
```npm link```

### Run app

Run the calculator: 

```
calc evaluate <math expression>
```
Ask for help:
```calc help
```
```calc help evaluate
```
